# Examination Results Dashboard – HTML & CSS

A single-page report card that shows Chidera’s Spring 2024 scores, GPA, grades and performance bars.  
No JavaScript, no images, just `index.html` and `style.css`.

---

## What you see
- **Header strip** – student name, semester, Total Subjects, Overall GPA, Average Score, Grade Distribution  
- **Subject table** – Mid-term, Final, Total, Grade (A/B) for 8 courses  
- **Performance metrics** – Highest, Lowest, Average, Range with mini progress bars  
- **Grade distribution** – count of A, B, C, D, F grades
